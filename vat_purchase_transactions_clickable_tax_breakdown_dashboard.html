Master Data consolidates the static reference modules into one sheet. Use the sub-sheets below for VAT Categories, ATC Master, and Supplier Master. Manual encoding is enabled for all three. This area is not filtered by date or month.
VAT Categories is a reference sub-sheet. Purchase Transactions use these codes to compute VAT; users upload/enter the code, not the VAT amount.
